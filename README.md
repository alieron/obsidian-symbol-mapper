# Symbol Mapper

Quickly search and insert special characters using fuzzy keywords.
This plugin lets you define a custom JSON mapping of special characters/strings to searchable keywords. Trigger the hotkey, type what you’re looking for, and insert the character into your note.

## Usage

1. Create your symbol map as a JSON object in the settings tab.
2. Open up the fuzzy search modal using `Ctrl + Shift + /`.
3. Search for your symbol.
4. Press `Enter` to paste the symbol at your cursor's position.
